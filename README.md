## 📘Project Overview

# Digitalized Campus Cafeteria System

## 📌 Introduction
The Digitalized Campus Cafeteria System is a C++ + MySQL project designed to modernize the university cafeteria by replacing manual processes with a digital solution. It automates customer registration, order placement, order tracking, and feedback collection, while enabling staff to manage orders in real-time.

## 🚩 Problem Statement
Traditional cafeteria workflows rely on paper slips and verbal communication, leading to:
- Redundancy in customer information  
- Inefficient order management  
- Human errors and delays  
- Lack of structured feedback  

## 🎯 Objectives
- Automate login/signup with validation  
- Enable dine-in and takeaway options  
- Provide staff with real-time order tracking  
- Allow deletion of completed orders  
- Collect structured customer feedback  
- Ensure data security and prevent duplicates  

## ⚙️ Functionalities
- User Management → Secure signup/login with phone number  
- Order Management → Place orders stored in MySQL  
- Order Tracking → Staff dashboard for active orders  
- Order Completion → Mark and delete completed orders  
- Feedback Module → Ratings and comments stored in DB  

## 🛠️ Methodology
1. User Registration & Login → Secure access with validation  
2. Order Placement → Structured interface, stored in DB  
3. Order Processing → Staff view linked to customer phone  
4. Order Delivery & Deletion → Active orders updated in real-time  
5. Feedback Collection → Ratings/comments analyzed for improvement  
6. Database Management → MySQL handles all records
